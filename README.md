# 📊 E-commerce & Marketing Performance Pipeline

An End-to-End data analytics project that integrates data cleaning, relational database modeling, and interactive visualization to measure marketing campaign performance.

---

## 🛠️ Tech Stack & Process / Flujo de Trabajo

### 1️⃣ Data Cleaning (Python)
* **Tools:** Python, Pandas.
* **Process:** Cleaned thousands of raw transactional rows, handled missing values, removed duplicates, and optimized date formats from `ventas_ecommerce.csv`.
* *Archivo:* `data_cleaning.py`

### 2️⃣ Database Modeling (SQL)
* **Tools:** SQL, SQLite / PostgreSQL structure.
* **Process:** Designed a relational schema with Fact and Dimension tables (`marketing_campaigns` and `ecommerce_sales`), optimizing primary/foreign keys to ensure data integrity.
* *Archivo:* `database_model.sql`

### 3️⃣ Interactive Dashboard (Power BI)
* **Tools:** Power BI, DAX.
* **Process:** Built an executive dashboard, filtered out refunds and organic traffic to calculate **Total Net Revenue ($773M)**, and created interactive slicers by campaign objective.
* *Archivo:* `marketing_performance_dashboard.pbix`

---

## 🇪🇸 [ES] Resumen del Proyecto

Este es un proyecto integral que conecta todo el flujo de trabajo de un Data Analyst. Se automatizó la limpieza de datos financieros con **Python**, se estructuró el modelo relacional en una base de datos **SQL**, y se diseñó un cuadro de mando dinámico en **Power BI** para la toma de decisiones estratégicas, identificando los canales de marketing más rentables (Affiliate y Paid Search).
